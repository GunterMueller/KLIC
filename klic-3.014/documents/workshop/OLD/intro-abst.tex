\section{構文と実行機構の基本}
\begin{itemize}
\item 簡単な例を使っての説明 (if-then)
\begin{verbatim}
not(In,Out) :- In=0 | Out=1.
not(In,Out) :- In=1 | Out=0.
\end{verbatim}
\begin{itemize}
\item 手続的意味 (informal)
\item 実行の過程
\begin{itemize}
\item リダクションの進行
\item データフロー同期による実行順変更
\end{itemize}
\item ユニフィケーション
\begin{itemize}
\item ガード: 同期とマッチング
\item ボディ: 具体化\\
(両方が具体値の場合は説明しない)
\end{itemize}
\end{itemize}
\item 構文の一般的説明\\
ヘッド引数は変数のみ\\
構造体は説明しない
\item 実行機構の説明\\
ゴール → サブゴールへの書換え
\item 補足事項
\begin{itemize}
\item 非決定性
\item 全節が失敗した時
\item 具体値同士のボディ・ユニファイ
\item otherwise (else)
\end{itemize}
\end{itemize}
